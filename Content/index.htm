Ivan's Test HTML Site

This is a sample MadCap Flare site. I publish it to my local drive that is synced to the Box cloud storage. I use fast.io to deploy this site on the web.

I use the MadCap Flare default HTML5 template with the side navigation. On this site, I show how to configure the target and publishing settings in Flare.
